# ElectroSureste Landing

Landing page built with Vite, React and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## GitHub Pages deployment

This repo is configured to deploy automatically to GitHub Pages from GitHub Actions.

Required one-time setup in GitHub:

1. Open the repository settings.
2. Go to `Settings > Pages`.
3. In `Build and deployment`, set `Source` to `GitHub Actions`.
4. Push to `main`.

The site is configured to work under:

`https://george28cs.github.io/electrosureste-landing/`
